# proverbium by [@michalbe](http://github.com/michalbe) #
Random, funny mix of Polish proverbs inspired by Słowa nad wyraz.

### How to use? ###
Install with:
```bash
 $ npm sudo install proverbium -g
```
and then...

```bash
 $ proverbium
```
